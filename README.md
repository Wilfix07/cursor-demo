# Cursor Demo Animation

Yon demo HTML/CSS/JavaScript ki montre yon cursor animé sou ekran

## Features
- **Cursor personnalisé** - kontour ble ki swivout mouvman souris
- **Effè ripple** - krounye vè yon klike
- **Trail particles** - partikèl ki swivout mouvman
- **Interactive dots** - 50 dots ki reagé woune klike

## Preview
https://wilfix07.github.io/cursor-demo/

## Lokal
```bash
python3 -m http.server 8000
# Ouvri http://localhost:8000/cursor-demo.html
```
